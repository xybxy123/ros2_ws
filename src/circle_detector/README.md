**5 个动态参数**：
1. `Blur Kernel` 模糊核
2. `Canny Thresh` 边缘阈值
3. `Hough Accum` 霍夫累加阈值
4. `Inner Dia(mm)` 内径
5. `Outer Dia(mm)` 外径

**先调图像 → 再调检测 → 最后定尺寸**

---

# 一、总原则
- **深度图噪点多**：不能太灵敏
- **圆环边缘清晰**：不需要太低阈值
- **尺寸用毫米**：最后再细调内径外径

---

# 二、逐个参数怎么调

## 1）Blur Kernel 模糊核
**作用：去噪、让边缘更连续**
- 深度图噪点多，**必须模糊**
- 只能是**奇数**：代码会自动修正

**调节顺序：第1个调**
- 深度图噪点多 → 往大拖
- 边缘太糊 → 往小拖

**推荐区间**
- 正常：**7 ~ 11**
- 噪点很大：**11 ~ 15**
- 图像很干净：**5 ~ 7**

---

## 2）Canny Thresh 边缘阈值
**作用：控制“什么才算边缘”**
- 越大：边缘越少、越干净
- 越小：边缘越多、噪点也多

**调节顺序：第2个调**
- 圆环出不来 → 改**小**（20→30）
- 杂边太多 → 改**大**（40→60）

**深度图推荐**
- 稳定值：**30 ~ 50**

---

## 3）Hough Accum 霍夫累加阈值
**最重要！决定圆能不能被检测出来**
- 越小：检测越多圆（包括噪声圆）
- 越大：只检测最清晰、最圆的

**调节顺序：第3个调**
- 圆环检测不到 → 改**小**（40→30）
- 一堆假圆、乱圆 → 改**大**（40→60→80）

**深度图推荐**
- 稳定值：**40 ~ 70**

---

## 4）Inner Dia(mm) 内径
## 5）Outer Dia(mm) 外径
**作用：最终筛选你要的圆环**
你目标是：
- 内径：**900 mm**
- 外径：**1200 mm**

**调节顺序：最后调**
- 先让圆环能稳定检测出来
- 再把内径外径设为：
  - `Inner = 900`
  - `Outer = 1200`

如果偶尔漏检，可以放宽一点：
- `850 ~ 1250`

---

# 三、标准调参步骤

## 步骤1：调模糊 Blur Kernel
- 拖到 **9**
- 看画面：圆环边缘平滑、没有颗粒感就行

## 步骤2：调 Canny Thresh
- 拖到 **40**
- 圆环轮廓断断续续 → 改小到 30
- 杂线太多 → 改大到 50

## 步骤3：调 Hough Accum
- 拖到 **50**
- 圆出不来 → 降到 40、35
- 乱圆太多 → 升到 60、70

## 步骤4：锁定内径外径
- `Inner Dia = 900`
- `Outer Dia = 1200`

完成！

---

# 四、典型问题快速修复表
1. **圆环检测不出来**
   → Hough Accum 改小
   → Canny 改小
   → Blur 稍微加大一点

2. **一堆假圆、噪声圆**
   → Hough Accum 改大
   → Canny 改大
   → Blur 加大

3. **圆环时有时无**
   → Hough Accum 调到 50~60 中间稳定值
   → Blur 固定 9

4. **明明是目标圆环却不显示**
   → 检查内径外径是否设为 900~1200
   → 深度是否在 300~2000mm 内
